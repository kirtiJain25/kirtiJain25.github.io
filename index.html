Sampling Large Graphs

• January 2018 - June 2018

Analysis of large scale graphs has been a challenging task due to computational overheads. Hence, use sampling to gather necessary information to estimate various graph properties of large graphs. Also studied Random Walk Sampling and performed thorough experimentation on several diverse data-sets spanning thousands nodes and edges.

Created a Graph Sampling python package containing various techniques which samples the original graph according to the sizes.
